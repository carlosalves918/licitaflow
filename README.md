# LicitaFlow v5

Plataforma de Gestão de Licitações Públicas.

## Funcionalidades
- Busca em tempo real via API PNCP
- Gestão de certames, habilitação e documentos
- Geração de declarações e propostas em PDF
- Controle de certidões com links para portais oficiais
- Calendário de prazos e notificações
- Análise de editais com IA

## Deploy na Vercel

1. Fazer push deste projeto no GitHub
2. Importar no Vercel: https://vercel.com/new
3. Framework: Create React App
4. Clicar em Deploy

## Desenvolvimento local

```bash
npm install
npm start
```
